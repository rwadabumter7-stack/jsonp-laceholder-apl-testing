# 🚀 JSONPlaceholder - API Automation & QA Testing Project

## 📌 Project Overview
This repository contains an advanced **API Testing & Automation Artifact** developed using **Postman**. The goal of this project is to implement automated test scripts to validate backend responses, ensure fast server performance, and design a customized visual layer for dry data payload verification.

---

## 🛠️ Automated Test Suite (Scope of Work)
Using Postman's JavaScript runtime, global assertions were created to validate endpoints on every request:
1. **HTTP Status Code Verification**: Enforcing `200 OK` and `201 Created` assertion blocks to verify server health.
2. **Performance Boundary Testing**: Automated runtime performance checks ensuring backend latency remains below `1500ms`.
3. **Data Visualization (UI Layer)**: Implemented Postman's visualization scripts to automatically convert structural JSON user payloads into fully formatted, human-readable HTML data tables.

---

## 📊 Deliverables Included
* **`My Collection.postman_collection RWAD`**: The official exported Postman collection containing the comprehensive request suites and verification logic, fully compatible with continuous integration (CI) workflows.
